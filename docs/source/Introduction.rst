Installation
------------

To Install Charmorph, you have two Options
Cloning Using Git CLI or Download them on `Releases <https://github.com/Upliner/CharMorph/releases/>`_

.. code-block:: console

   git clone https://github.com/Upliner/CharMorph --recursive
